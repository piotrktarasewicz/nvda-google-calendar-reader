Privacy Policy

NVDA Google Calendar Reader is an add-on for the NVDA screen reader that allows users to hear events from their Google Calendar using keyboard shortcuts.

Data Access

The application requests access to Google Calendar events using the official Google Calendar API. Access is strictly limited to read-only calendar data.

The application does not create, edit, or delete calendar events.

Purpose of Data Use

Calendar data is used solely to read event information to the user through NVDA speech output.

Data Storage

Calendar event data is not stored on any external servers operated by the developer.

Authentication tokens may be stored locally on the user's computer in order to allow continued access to Google Calendar without requiring repeated logins.

Data Sharing

No calendar data is sold, shared, or transmitted to third parties.

All processing occurs locally on the user's device.

Security

The application uses Google's OAuth authentication system and the official Google Calendar API.

Revoking Access

Users may revoke the application's access to their Google account at any time through their Google Account security settings.
