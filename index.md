# Google Calendar Reader

Google Calendar Reader is an NVDA add-on for Windows that allows blind and visually impaired users to access their Google Calendar through keyboard shortcuts and speech.

## What the add-on does

Google Calendar Reader allows the user to:

- sign in to Google Calendar,
- read events for today and the following days,
- choose which calendars should be included in speech output,
- switch between short and full reading modes,
- hear when an event is currently in progress.

The add-on is designed for efficient and accessible calendar reading in NVDA.

## Google account access

The add-on uses Google OAuth sign-in through the user's web browser.

It requests access to Google Calendar in read-only mode only.

The add-on does not create, edit, or delete events.

## Local operation

Google Calendar Reader is designed to work locally on the user's computer.

Calendar data is used only to provide spoken output through NVDA. The developer does not operate a server that stores users' calendar data.
