NVDA Google Calendar Reader

NVDA Google Calendar Reader is an add-on for the NVDA screen reader on Windows. It allows users to quickly hear upcoming events from their Google Calendar using keyboard shortcuts.

What the application does

The add-on lets users sign in to their Google account and read events from selected Google Calendars.

It works entirely locally on the user's computer and uses the official Google Calendar API with read-only access.

Main features

Keyboard shortcuts allow users to quickly hear events for today and the following days without opening the Google Calendar website.

Users can also choose which calendars should be included in the spoken results.

Privacy

The application only requests read-only access to the user's Google Calendar. Calendar data is not transmitted to any external servers controlled by the developer.
